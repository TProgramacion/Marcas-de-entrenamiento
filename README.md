# Marcas de Entrenamiento en C#

Aplicación de consola desarrollada en C# que permite gestionar sesiones de usuarios y registrar sus marcas de entrenamiento.  
Fundamentos de Programación (DAW).

## Funcionalidades

- Registro de usuarios (correo y contraseña, evitando duplicados)  
- Inicio de sesión de usuarios  
- Registro de entrenamientos (distancia y tiempo)  
- Listado de entrenamientos por usuario  
- Vaciar la lista de entrenamientos  
- Cierre de sesión  
- Interfaz mediante menú iterativo en consola

## Tecnologías utilizadas

- Lenguaje: C#  
- Entorno: Visual Studio Code  

## Objetivos de aprendizaje

- Uso de comandos de C#  
- Manejo de variables y estructuras de control  
- Creación y gestión de listas  
- Trabajo con objetos  
- Desarrollo de una aplicación interactiva mediante menú iterativo
